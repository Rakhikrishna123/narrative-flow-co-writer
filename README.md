# narrative-flow-co-writer
## Overview
Narrative Flow Co-Writer is an AI-assisted storytelling application built using Streamlit and TinyLlama.  It enables interactive story generation based on user prompts.
## Objective
To explore lightweight Large Language Model (LLM) integration for human-AI collaborative storytelling.
## Architecture
User Input → Prompt Processing → TinyLlama Model → Generated Narrative → Streamlit UI
## TechnologiesUsed
- Python
- - Streamlit
- - TinyLlama (Local LLM)
- - Prompt Engineering
## Features
-Real-time story generation
- Adjustable tone and genre
- - Interactive chat-based interface
- - - Session state management
## Research Relevance
This project investigates:
- Lightweight LLM usability
- - Prompt-based narrative control
- - Human-AI collaborative creativity
